## Northwest

Northwest is a game created on the technologies of the web (React and ASP.NET). I decided to use these odd technologies for this game because all the interactions in the game occur in the UI, whereas game engines are more well-suited for graphics rendering and audio processing. We have to see this a digital board-game. 

For the game's pitch (it's pretty good) [View Repo] (https://github.com/AetherFire23/NorthwestV2-server.git)
